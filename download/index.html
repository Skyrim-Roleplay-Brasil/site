📋 Como instalar

  1. Baixe o instalador (.exe, 85 MB).
  2. Execute e siga o wizard (Next → Install → Finish). Pode aparecer alerta do Windows Defender — clique "Mais informações" → "Executar mesmo assim". É falso positivo conhecido.
  3. Abra o launcher. Ele detecta seu Skyrim Special Edition via Steam. Se não detectar: abra o Steam uma vez antes.
  4. Entre com Discord. O launcher valida a whitelist automaticamente via seu Discord ID.
  5. Validação SHA256 na primeira execução. Se falhar, o launcher oferece "Reparar via Steam".
  6. Pronto: clique em "Jogar". Você entra direto no servidor.

⏱ Tempo total: ~6 min em banda larga.

🔧 Resolução de problemas

🛡️ Antivírus bloqueia

Falso positivo conhecido. Adicione C:\Users\<seu>\AppData\Local\Programs\SkyrimRP-Launcher\ às exceções.

🔍 Skyrim não detectado

Abra o Steam uma vez antes. O launcher procura pelo registro Steam.

🔐 Erro de SHA256

Arquivos do Skyrim corrompidos. No Steam: Propriedades → Arquivos locais → "Verificar integridade".

🎙️ Voz LiveKit não conecta

Firewall corporativo ou VPN bloqueando UDP porta 7880. Tente em rede doméstica.

💥 Crash ao entrar

Mod em conflito. Use "Restaurar snapshot" no menu Ajuda.
